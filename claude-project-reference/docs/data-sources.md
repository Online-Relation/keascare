# Datakilder

## Tilbudsportalen
Bruges til at finde bosteder/tilbud.

Mulige datafelter:

- Navn
- Adresse
- Kommune
- Driftstype
- Pladser

## STPS
Bruges til tilsynsrapporter og signaler.

Mulige datafelter:

- Rapporttitel
- Rapportdato
- Rapportlink
- Konklusion
- Fund-kategori

## Monday API
Bruges til CRM-match og lead-oprettelse.

Mulige datafelter:

- Kunde/lead-status
- Pipeline
- Ansvarlig
- Sidste aktivitet

## Danmarks Statistik
Bruges kun til overordnet behov pr. kommune.

Mulige datafelter:

- Borgere i botilbud pr. kommune
- §107 / §108 hvis tilgængeligt
