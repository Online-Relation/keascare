# STPS parsing og fund-kategorier

## Princip
Systemet må ikke opfinde en alvor-score. Det skal oversætte STPS' egne konklusioner.

## Mapping

| Tekst fundet i rapport | Systemlabel |
|---|---|
| Kritiske problemer | Kritiske problemer |
| Større problemer | Større problemer |
| Mindre problemer | Mindre problemer |
| Ingen problemer | Ingen problemer |
| Ikke fundet | Ukendt |

## AI
AI må senere bruges til:

- Opsummering
- Fokusområder
- Relevante KeasCare-ydelser
- Salgsvinkel

AI må ikke være primær kilde til alvorlighed.
